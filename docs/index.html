    Je suis un absolute!

    Box 4

    La propriété text-align décrit la façon dont un texte en ligne (inline) est alignée dans le bloc de son élément parent. text-align ne contrôle pas l'alignement des éléments en bloc, uniquement celui du contenu en ligne.

    Fonctionne uniquement avec display: inline ou inline-block

    ! Doit être appliqué à l'élément parent !

                        .box-container1 {
  text-align: center;
}
.box1 {
  display: inline-block;
}

    Vous pouvez mettre la valeur auto aux propriétés margin-right et margin-left pour centrer horizontallement un élément à l'intérieur de son conteneur. L'élément prendra la largeur que vous avez spécifié et l'espace restant sera séparé équitablement entre les deux marges.

    Attention à ne pas oublier de définir une largeur pour l'élément à centrer, sinon il ne pourra logiquement pas être centré (il occupera toute la largeur disponible, comme tous les blocs par défaut).

                         .box2 {
  margin: 0 auto;             
}

    Pour centrer un element absolute il suffit d'utiliser

                         .box3 {
  left: 50%;
  transform: translateX(-50%);
}

    Centrer horizontalement sur le même axe avec la propriété justify-content.

    Doit être appliqué à l'élément parent !

                         .box-container4 {
  display: flex;
  align-items: center;
}

    Pour aligner verticalement un mot ou une ligne de texte au sein de son conteneur, il suffit de jouer avec la propriété line-height et de lui indiquer la même valeur que son parent.

                            .box1 {
  width: 150px;
  height: 100px;
  line-height: 100px; /* hauteur de ligne 
  (identique) */
}

    On utilise ici la propriété display: flex sur l'élément parent et on défini les margin en auto

                             .box-container2 {
  display: flex;            
}
.box2 {
  margin: auto;
}

    La puissance de Flexbox est également dans sa capacité à centrer des éléments verticalement, voici la même application des flexbox que precédemment, mais avec les éléments flex également centrés verticalement

                             .box-container3 {
  display: flex;
  justify-content: center;
  align-items: center;
}

    On utilise ici le même principe que le centrage horizontal d'un absolute, mais avec translateY()

                             .box4 {
  left: 50%;
  transform: translateY(-50%);
}
  • La position des éléments

    Box 1

    Box 2

    static est la valeur par défaut de tous les éléments. Un élément avec position: static; n'est positionné d'aucune manière spéciale. Un élément static est dit non positionné et un élément avec une propriété position ayant une valeur autre que static est dit positionné.

                        .box2 {
  position: static;
}

    relative se comporte de la même façon que static sauf si vous ajoutez quelques propriétés en plus.

                        .box2 {
  position: relative;
  top: -50px;
  left: 0;
}

    Un élément positionné en fixed est positionné par rapport a la fenêtre du navigateur, ce qui signifie qu'il reste toujours à la même place même si la page défile. De la même manière qu'avec un élément positionné en relative, nous pouvons utiliser les propriétés top, right, bottom et left.

                        .box2 {
  position: fixed;
  bottom: 0;
  left: 0;
}

    absolute est la valeur la plus délicate. absolute se comporte comme fixed sauf que son positionnement est relatif à l'élément parent positionné le plus proche au lieu d'être relatif à la fenêtre du navigateur. Si un élément positionné en absolute n'à aucun élément parent positionné, il utilise le corps du document et se déplace avec le document au défilement de la page. N'oubliez-pas qu'un élément "positionné" est un élément dont la position est tout sauf static.

                        .box2 {
  position: absolute;
  top: 0px;
  left: 0px;
}
